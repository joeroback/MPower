# Telemetry Overlay

The minimum CSV headers required by Telemetry Overlay when importing a Harry's Laptimer formatted CSV:
 * 'INDEX'
 * 'LAPINDEX'
 * 'DATE'
 * 'TIME'
 * 'TIME_LAP'
 * 'LATITUDE'
 * 'LONGITUDE'
 * 'SPEED_KPH'
 * 'SPEED_MPH'
 * 'HEIGHT_M'
 * 'HEIGHT_FT'
 * 'HEADING_DEG'
 * 'DISTANCE_KM'
 * 'DISTANCE_MILE'
 * 'ACCELERATIONSOURCE[CALCULATED/MEASURED/UNDEFINED]'
 * 'LINEALG'
 * 'LATERALG'
 * 'GEAR'
 * 'RPM'
 * 'THROTTLE'
 * 'BRAKEPRESSURE'
 * 'STEERINGANGLE'
